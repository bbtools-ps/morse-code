# Morse code translator

Translate to Morse code alphabet and vice versa.

This project uses [React.js](https://reactjs.org/).

## Project setup

To install all dependencies:

`npm install`

To run the project in the local development mode:

`npm start`

To make the build:

`npm run build`

To start the tests:

`npm run test`
