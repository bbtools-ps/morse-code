# Morse code translator

Translate to Morse code alphabet and vice versa.

This project uses:

- [PNPM](https://pnpm.io/)
- [React.js](https://reactjs.org/).

## Project setup

To install all dependencies:

`pnpm install`

To start the project in the local development mode:

`pnpm dev`

To make the build:

`pnpm build`

To start the tests:

`pnpm test`
